## Repository Policy

- No PDFs from standards or papers.
- No copied figures from papers unless permission is explicit.
- No media files, DASH segments, generated logs, CSVs, zips, `.venv`, caches, IDE folders, or local configs.
- Scientific claims must trace to source cards, matrices, or future implementation specs.
