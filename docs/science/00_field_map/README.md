# Field Map

This folder defines the scientific frame for Phase 2 before any ABR baseline is implemented.

It covers:

- DASH terminology and client-driven adaptive streaming context.
- Survey-level ABR taxonomy.
- Recent HAS and end-to-end video streaming reviews.
- Local UGR-related streaming traffic work.
- Scope decisions for what belongs in the initial TFG baseline set.

The field map is not an implementation plan by itself. It feeds the baseline selection work in `../01_baselines/` and the thesis-memory material in `../07_memory/`.
